# Quasar UI Builder

VS Code 기반의 Quasar UI 화면 개발 툴을 만들기 위한 프로젝트 작업 공간입니다.

## 개발 단계

1. 화면 JSON -> Vue 파일 생성기
2. 컴포넌트 팔레트 + 속성 패널
3. 드래그앤드롭 화면 배치
4. 그리드, 팝업, 공통코드, API 자동연결
5. VS Code Extension 통합

## 참고 방향

xFrame, WebSquare, eXbuilder의 화면 설계 방식, 컴포넌트 속성 편집, 데이터셋/공통코드 연결, 화면 산출물 생성 흐름을 참고합니다.
